### 📈 Mutual Fund Analysis Dashboard

### 📌 Project Overview

The Mutual Fund Analysis Dashboard is an interactive solution built using Python, Excel, and Power BI.
It processes data from over 2,500+ mutual fund schemes and identifies the top 30 best-performing funds based on returns, risk, expense ratio, and fund age.
The dashboard enables investors and analysts to make data-driven decisions by comparing fund performance with interactive visuals.

---

### 🚀 Key Features

📊 Interactive Power BI Dashboard showcasing top-performing mutual funds.

🔍 Data Cleaning & Preprocessing of 2,500+ schemes for accuracy.

📈 Scoring Model combining return %, risk, expense ratio, and fund age.

🏆 Top 30 Funds Selection with risk-adjusted returns.

📉 Expense Ratio & Fund Age Analysis for better investment evaluation.

📂 Python Automation for normalization & scoring.
